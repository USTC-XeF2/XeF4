# XeF4
 A comprehensive QQ group bot

## 所需环境
- Python 3.9+
- NoneBot2（同时安装 `nb-cli` 框架）
- 驱动器（使用 `nb driver install` 安装）
  - fastapi
  - websockets
- 适配器（使用 `nb adapter install` 安装）
  - onebot
  - minecraft
- 插件（使用 `nb plugin install` 安装）
  - nonebot-plugin-group-config
  - nonebot-plugin-follow-withdraw
